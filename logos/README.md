# 📁 Public Logos Folder

Place your logo images here and reference them directly in your code.

## How to Use:

1. **Add your images** to this folder:
   - `nexus-logo.png`
   - `excel-logo.png`
   - etc.

2. **Reference in JSX/TSX**:
   ```tsx
   <img src="/logos/nexus-logo.png" alt="Nexus Logo" />
   ```

3. **Reference in CSS**:
   ```css
   background-image: url('/logos/nexus-logo.png');
   ```

## Recommended Image Formats:
- **PNG**: For logos with transparency
- **SVG**: For scalable logos (best quality)
- **WEBP**: For photos (smaller file size)

## Tips:
- Use descriptive filenames: `nexus-logo.png` instead of `logo1.png`
- Optimize images before adding (use tools like TinyPNG)
- Keep file sizes under 200KB when possible
